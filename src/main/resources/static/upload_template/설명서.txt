[현장조사서 일괄 업로드 가이드]

이 파일은 현장조사서를 일괄 업로드하기 위한 양식입니다. 아래 지침에 따라 데이터를 입력하고, 파일을 준비해 주세요.

1. 엑셀 파일 (Batch_Upload_List.xlsx)
   - 이 파일에 현장조사서 정보를 입력하세요.
   - **필수 입력 항목**:
     - **사실상 지목:** 토지의 실제 사용 용도
     - **사실상 면적:** 실제 사용 면적
     - **소유자:** 토지 소유자
     - **주사용 실태:** 토지의 주 사용 목적
     - **조사자 소속:** 조사자 소속 기관명
     - **조사일:** 조사가 수행된 날짜 (YYYYMMDD 형식)
     - **조사자 성명:** 조사자 이름
     - **확인자:** 조사서를 확인한 담당자 이름

2. 폴더 구조
   - 각 조사서는 고유 ID(예: GRNDS00001)로 구분됩니다.
   - 아래와 같은 폴더 구조를 유지하세요:

     GRNDS00001/
     ├── aerial_photos/ (항공사진)
     └── site_photos/ (현장사진)

3. 사진 파일
   - 각 폴더에 필요한 사진 파일을 넣어 주세요.
   - 사진 파일명은 가능한 영어로 작성해 주세요.

4. 파일 및 폴더 이름
   - 제공된 이름을 변경하지 말고 그대로 사용해 주세요.

5. **중요**: 다른 사용자가 현장조사서를 먼저 등록할 경우, 제공된 폴더명(ID)이 이미 사용 중일 수 있습니다.
   - 만약 업로드 시 '폴더명이 이미 사용 중입니다'라는 메시지가 나타나면, 새로운 양식을 다운로드하여 최신 ID로 다시 등록해 주세요.

6. 업로드 방법
   - 모든 내용을 입력한 후, ZIP 파일로 압축하여 업로드하세요.
   - 잘못된 내용이 있으면 시스템에서 오류 메시지가 표시됩니다.

감사합니다.
